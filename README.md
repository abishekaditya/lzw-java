####How to Use
1.Download LZW.jar.


2.Copy LZW.jar to your file and open it.


3a.Compress

	-Input: filename
	-Output: any easy to remember name
	-Press Compress button
	
	
3b.Decompress
	
	-Input: compressed filename
	-Output: any name with correct extension
	-Press Decompress button
